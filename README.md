#  React Todo List App

A simple and responsive Todo List application built using React.js.  
This project helps users manage daily tasks easily with add and delete functionality.

---

##  Features

- ➕ Add new tasks  
- ❌ Delete tasks  
- 🎨 Clean and responsive UI design  
- ⚡ Fast React state management  

---

## 🛠️ Tech Stack

- React.js  
- JavaScript (ES6+)  
- HTML5  
- CSS3
